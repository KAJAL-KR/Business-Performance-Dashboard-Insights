# 📊 Business Performance Analysis & Reporting

## 🔍 Overview  
Analyzed business transaction data (1,194 records from 2020–2025) using Python and SQL to uncover insights on sales performance, product trends, and profitability.  

This project simulates real-world **Junior Data Analyst responsibilities**, focusing on data cleaning, SQL-based analysis, and visualization.

---

## 📌 Problem Statement  
Businesses often struggle to extract meaningful insights from transactional data, making it difficult to identify profitable products, payment trends, and regional performance.

---

## 🎯 Objectives  
- Analyze sales and profit trends  
- Identify top-performing products and categories  
- Evaluate payment methods and regional performance  
- Generate insights to support business decision-making  

---

## ⚙️ Approach  
1. Data loading and exploration  
2. Data cleaning and preprocessing  
3. SQL-based analysis (operational & product insights)  
4. Data visualization (charts & trends)  
5. Exporting final business report  

---

## 📈 Key Insights  
- 💳 Debit Card is the top payment method by usage and profit  
- 🛍️ Markers & Tables are the highest revenue-generating sub-categories  
- 📍 Florida & New York lead in overall profit  
- 📊 Clear trends observed in sales and product performance  

---

## 🛠 Tools & Technologies  
- Python (Pandas, Matplotlib, Seaborn)  
- SQL (SQLite3 – in-memory queries)  
- Data Analysis & Visualization  

---

## 💡 Skills Demonstrated  
- SQL querying and data analysis  
- Data cleaning and preprocessing  
- Business insight generation  
- Data visualization and reporting  

---

## 📁 Dataset  
- 1,194 transactions  
- 12 columns  
- Time period: 2020–2025  

---

## 🚀 Conclusion  
This project demonstrates how combining Python and SQL can transform raw business data into actionable insights, enabling better decision-making and performance tracking.
